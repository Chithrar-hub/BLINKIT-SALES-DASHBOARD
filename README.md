# BLINKIT-SALES-DASHBOARD
# Blinkit Sales Analysis Dashboard | Power BI Project

1 Project Introduction

The Blinkit Sales Analysis Dashboard is a Power BI project designed to analyze retail sales performance, customer purchasing patterns, outlet performance, and product category insights across different outlet types and locations.

This dashboard transforms raw grocery sales data into interactive business intelligence visuals that help stakeholders monitor revenue trends, product demand, outlet efficiency, and operational performance.

---

2 Project Objective

The main objectives of this project are:

* Analyze overall sales performance across outlets
* Monitor average sales and product distribution
* Identify high-performing outlet locations and outlet types
* Compare sales contribution by item categories
* Track outlet establishment growth trends
* Build an interactive and user-friendly retail analytics dashboard

8 Key Insights

* Medium-sized outlets generate the highest sales contribution.
* Tier 3 outlets outperform Tier 1 and Tier 2 locations in total revenue.
* Fruits, snack foods, and household items contribute the highest sales.
* Regular fat products account for the majority of total sales.
* Supermarket Type1 outlets handle the highest number of items sold.
* Outlet establishment growth peaked significantly around 2018.
